# A more responsive Google Search page
User-style that makes the Google Search page more responsive and easier to use on split screen.
## Preveiw
<img style="width: 100%;" src="https://raw.githubusercontent.com/Andreto/google-search-responsiveness/main/prev.png"></img>
## Install
1. Install a user-style handler like [Stylus](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne).
2. Open [google-search-responsiveness.user.css](https://github.com/Andreto/google-search-responsiveness/raw/main/google-search-responsiveness.user.css)
